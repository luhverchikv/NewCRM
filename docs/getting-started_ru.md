# Начало работы

## Установить
Клонировать, копировать окружение и синхронизировать его с uv.
```баш
клон git https://github.com/your-org/aiogram-bot-template.git
cd aiogram-bot-шаблон
cp .env.пример .env
УФ-синхронизация
10|```
11|Отредактируйте `.env`, указав свой `TG__BOT_TOKEN` и идентификаторы администратора.
12|
13|## Бежать
14|По умолчанию используется опрос;Режим веб-перехватчика контролируется `TG__WEBHOOK_USE`.
15|```баш
16|uv запустить источник python/__main__.py
17|```
18|
19|## Первый хендлер
20|Добавьте команду существующему пользовательскому маршрутизатору, и он будет зарегистрирован автоматически.
21|`source/telegram/handlers/user/commands.py`:
22|```питон
23|из команды импорта aiogram.filters
24|из сообщения импорта aiogram.types
25|
26|@user_commands_router.message(Команда("ping"))
27|async def ping(сообщение: Сообщение) -> Нет:
28|дождитесь сообщения.ответ("понг")
29|```
30|`source/telegram/handlers/user/__init__.py`:
31|```питон
32|router.include_router(user_commands_router)
33|```
