4|```текст
5|aiogram-bot-template/
6|├── документы/
7|├── миграции/
8|├── скрипты/
9|├── источник/
10|│ ├── конфигурация/
11|│ ├── база данных/
12|│ ├── завод/
13|│ ├── инфраструктура/
14|│ ├── локали/
15|│ ├── услуги/
16|│ ├── телеграмма/
17|│ └── утилиты/
18|├── тестов/
19|├── alembic.ini
20|├── docker-compose.dev.yml
21|├── docker-compose.yml

25|
26|## Папки
27|- `docs/` краткие руководства и справочная документация.
28|- `source/config/` pydantic настройки и загрузка окружения.
29|- модели `source/database/`, репозитории, UnitOfWork.
30|- Уровень бизнес-логики `source/services/`.
31|- обработчики `source/telegram/`, маршрутизаторы, фильтры, диалоги.
32|- `source/factory/` подключение бота/диспетчера/приложения/контейнера.
33|- Кэш `source/infrastructure/` и внешняя интеграция.
34|- `source/utils/` регистратор, i18n, помощники.
35|- `migrations/` Скрипты миграции Alembic.
36|- `scripts/` Помощники CLI для задач эксплуатации.
37|
38|## Ключевые файлы
39|- Примеры значений конфигурации `.env.example`.
40|- `alembic.ini` Конфигурация Alembic.
41|- `docker-compose.yml` стек создания продукции.
42|- `docker-compose.dev.yml` локальный стек БД/Redis.
43|- точка входа в приложение `source/__main__.py`.
