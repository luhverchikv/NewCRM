3|## Обязательные переменные
4|Установите их в `.env` перед запуском;Настройки загружают их все.
5|```окр.
6|TG__BOT_TOKEN=***
7|TG__ADMIN_IDS=[5252216460]
8|TG__WEBHOOK_USE=Ложь
9|TG__WEBHOOK_PATH=/телеграмма
10|
11|WEBHOOK__URL=https://example.com
12|ВЕБХУК__ХОСТ=0.0.0.0
13|ВЕБХУК__PORT=8443
14|WEBHOOK__PATH=/вебхук
15|WEBHOOK__SECRET=long_r...cret
16|
17|DB__HOST=дб
18|DB__PORT=5432
19|DB__USER=по умолчанию
20|DB__ПАРОЛЬ=***
21|DB__NAME=telegram_bot_template
22|
23|REDIS__HOST=редис
24|REDIS__PORT=6379
25|REDIS__USER=по умолчанию
26|REDIS__ПАРОЛЬ=***
27|REDIS__DB=0
28|```
29|
30|## Необязательные переменные
31|`ENVIRONMENT` по умолчанию имеет значение `production`.
32|```окр.
33|ОКРУЖАЮЩАЯ СРЕДА=развитие
34|```
